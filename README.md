# Hinterland Vegetables
Hinterland Vegetables is the second in the Hinterland series of mods that aims to add a variety of new plants to the world. All are edible, though some will need to be processed in some way to avoid poisoning you. The eventual goal is full Biomes and Expanded Foods compatibility.

## Crops
- Cucumber
- Tomato
- Jalapeno
- Habanero
- Ginger
- Rhubarb
- Plantain
- Leek
- Butternut Squash
- Potato
- Asparagus

## Wild
